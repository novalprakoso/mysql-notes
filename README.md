# 📘 MySQL Learning Notes

Repository ini berisi rangkuman pembelajaran **MySQL** yang saya pelajari secara mandiri melalui berbagai sumber, khususnya dari kanal YouTube **Indonesia Belajar**.
https://www.youtube.com/@belajaridn

Dokumentasi ini dibuat sebagai bagian dari pengembangan skill di bidang **Data Analyst / Data Engineer**, sekaligus sebagai portfolio pembelajaran.

## 📚 Materi yang Dipelajari

* SQL 00: Koneksi Database MYSQL MARIADB
* SQL 01: Seleksi kolom dengan SELECT
* SQL 02: Pengurutan Data ORDER BY
* SQL 03: Seleksi Baris WHERE
* SQL 04: Alias Untuk Kolom dan Tabel, CONCAT, CONCAT_WS
* SQL 05: INNER JOIN, ON, USING
* SQL 06: Pengelompokkan Data GROUP BY
* SQL 07: Seleksi Pengelompokan Data dengan HAVING
* SQL 08: Membuat Table atau Data Definition Language DDL
* SQL 09: INSERT, UPDATE, DELETE-DATA MANIPULATION LANGUAGE (DML)
* SQL 10: SUBQUERY
* Studi kasus sederhana
